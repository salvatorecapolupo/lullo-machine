La macchina combinatoria

Questa applicazione simula la macchina combinatoria ideata nel XIII secolo dal filosofo e teologo Raimondo Lullo (Ramon Llull). Lullo immaginò una serie di dischi concentrici che potevano ruotare per combinare concetti fondamentali della filosofia e della teologia.

Ogni disco contiene nove elementi. Quando i dischi ruotano, gli elementi allineati sotto l'indicatore generano una proposizione logica.

Con tre dischi da nove elementi ciascuno si ottengono 9 × 9 × 9 = 729 combinazioni possibili.

Raimondo Lullo

Raimondo Lullo (1232–1316) fu un filosofo, teologo e scrittore catalano. È considerato uno dei pionieri del pensiero combinatorio e della logica applicata alla conoscenza.

Il suo progetto filosofico più famoso è l'Ars Magna, un metodo per generare e dimostrare verità attraverso combinazioni di concetti fondamentali.

Per realizzare questo metodo Lullo progettò diagrammi e macchine composte da ruote rotanti. Ruotando le ruote era possibile combinare idee e produrre proposizioni che potevano essere discusse, analizzate o dimostrate.

Molti studiosi considerano queste macchine combinatorie un lontano antenato dei sistemi logici formali e, in senso molto generale, dei moderni sistemi computazionali.

Significato delle lettere

Nella macchina lulliana i concetti sono rappresentati da lettere. La lettera A indica il principio centrale, cioè Dio, origine di tutte le perfezioni. Le altre lettere rappresentano le cosiddette "dignità divine".

Lettera Significato
ADio / principio assoluto
BBontà
CGrandezza
DEternità
EPotenza
FSaggezza

IVerità
KGloria

Simulazione in C++ (versione semplificata)

Questo programma C++ mostra una versione molto semplice della macchina: tre array circolari rappresentano i tre dischi. Ruotando gli indici si genera una nuova proposizione. 


#include <iostream>
#include <string>
using namespace std;

const int N = 9;

string dignita[N] = {
"Bontà","Grandezza","Eternità","Potenza","Saggezza",
"Volontà","Virtù","Verità","Gloria"
};

string relazioni[N] = {
"differisce da","concorda con","è contraria a",
"è principio di","è mezzo tra","è fine di",
"è maggiore di","è uguale a","è minore di"
};

string soggetti[N] = {
"Dio","Angelo","Cielo","Uomo",
"Immaginativa","Sensitivo",
"Vegetativo","Elementale","Strumentale"
};

int main() {

int a=0,b=0,c=0;

cout << "Macchina di Lullo\n";

while(true){

cout << "\n";
cout << dignita[a] << " "
     << relazioni[b] << " "
     << soggetti[c] << endl;

cout << "1 ruota disco 1\n";
cout << "2 ruota disco 2\n";
cout << "3 ruota disco 3\n";
cout << "0 esci\n";

int scelta;
cin >> scelta;

if(scelta==1) a=(a+1)%N;
if(scelta==2) b=(b+1)%N;
if(scelta==3) c=(c+1)%N;
if(scelta==0) break;

}

}
